# ClubManagement

## 📌 프로젝트 개요

- **프로젝트명**: ClubManagement  
- **목적**:  
  호회 운영 시, 모임 참석 및 회비 납부 정보를 수기로 관리하면서 발생하는 불편함을 해결하고자,  
  출석률과 회비 납부 현황을 체계적으로 확인할 수 있는 **모임 관리 프로그램**을 개발합니다.  

- **주요 기능**:
  - 회원 정보 관리
  - 정기/비정기 모임 출석 관리
  - 회비 납부 및 미납 내역 관리
  - 게임 기록 및 스코어보드 관리
